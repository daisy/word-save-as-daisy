OpenXML/DAISY Translator Add-in for Microsoft Word

Thank you for downloading the OpenXML/DAISY Translator Add-in for Word. The add-in enables you to translate Microsoft Word documents to the DAISY specification format documents.The DAISY Translator project is hosted on sourceforge.net

Installing the Add-in

This section details on the steps that should be followed while installing an Add-In: Before installation user should close all office applications, as because it may cause some conflict.

Step: 1 Click on the DaisyAddinForWordSetup.exe. File installation will start and user will be presented a window, which says welcome to DAISY Translator setup wizard, giving users option to either Cancel or choose next. Click on Next button or just hit Enter.

Step 2: Now user will be presented a window, which shows “License Agreement”. In this step user need to select the check box “I accept the terms in the License Agreement” and hit enter or click on Next button.

Step 3: In this step user should select an installation scope and hit enter or click on Next button.

Step 4: Now user will be presented a window, which says “Select Installation Folder”. Hit enter or click on Next button. Users are given option to browse for folder either by hitting on change button or by entering (Alt + C) on the keyboard.

Step 5: Now user will be presented a window, which says “Confirm Installation”. Hit Enter or click on Install button.

Step 6: Now a progress window appears showing the progress of the installation.

Step 7: Now user will be presented a window, which says “Completed the DAISY Translator Setup” with an option to open the Help file. Hit Enter or click on Finish button.By selecting the “Show Help” option, this opens up help files giving instructions on installing Add-In and converting word documents to Daisy format.

Troubleshooting

If you don't see the "Save As DAISY" entry in the Microsoft Word File menu, this might be due to missing features in Word (see Software Requirements section) or a failure during the installation of the Word Primary Interop Assemblies.

For more information on Troubleshooting, download troubleshooting guidelines from sourceforge.net.

Features Supported

  • Dublin Core Metadata
  • UID metadata
  • Front matter, Book matter and rearmatter mapping
  • Levels and Headings
  • Paragraphs
  • Tables
  • Lists
  • Note’s and Note References
  • Emphasis and Strong
  • Superscript and Subscript
  • Images
  • Validity
  • Captions
  • Page Numbers
  • Sidebar
  • Abbreviations and Acronyms
  • Language Detection
  • Blockquotes
  • Anchor
  • Bdo (Bidirectional object)
  • Citation
  • Table of Contents
  • Custom styles
  • Helper Wizard
  • MathML (Support for Mathematics)
  • Multiple OOXML documents
